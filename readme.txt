1. Главная Страница
Содержимое:
	Выбор перехода
Переходы:
	Счета
	Клиенты
	Отделения
	Транзакции
2. Счета
Содержимое:
	Форма для поиска счетов с фильтром
	Кнопка добавить счёт
	 У каждого закрыть .
Переходы: главная страница, добавление счёта

5. Транзакции
Содержимое:
 	Форма для поиска транзакций с фильтром
	добавить транзакцию.
	Переходы: главная страница, добавление транзакции
6.   Отделения
Содержимое:
	Поиск отделения с фильтром(имя, адрес) 
	добавить отделение
	удалить отделение
	Переходы: главная страница, добавить отделение, изменить информацию об отделении
7. Клиенты
Содержимое:
	Форма для поиска клиентов с фильтром (по  имени, контактной информации, виду счёта, отделению, датам  регистрации)
	 Добавить клиента
	 У каждого изменить и удалить.
	 ереходы: главная страница, добавить клиента.

Схема использования:
1 Получение списка клиентов, в т.ч. по типам, видам счетов в заданном интервале времени и пр.
	Главная страница-> клиенты (вид счета из таблицы счёта, d)
2 Получение списка счетов по их видам, списаниям/начислениям за заданный период
	Главная страница -> счета (вид из таблица счетов, списания/начисления из таблицы транзакций)
3 Получение списка отделений, в. т.ч. по клиентам, счетам и пр.
	Главная страница-> отделения (если ищем по имени адресу)
	Главная страница -> клиенты
	Главная страница -> счёта
4. Оформление списания/начисления, включая автоматический учет процентов
	Главная страница -> Транзакции ->  добавить транзакцию
5. Заведение счета и его закрытие, чтение данных о нем
	Главная страница->Счета(добавить счёт, -> cчет закрыть , читаем из выведенной таблицы)
6. Добавление и удаление данных о клиенте, чтение и редактирование данных о нем
	Главная страница->Клиенты-> (добавить, -> клиент Информация / редактировать / удалить)
7.Добавление и удаление отделения, чтение и редактирование данных о нем
Главная страница -> Отделения (добавить, ->отделение чтение / редактировать / удалить )

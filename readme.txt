1. Главная Страница
Содержимое:
	Выбор перехода
Переходы:
	Счета
	Клиенты
	Отделения
2. Счета
Содержимое:
	Форма для поиска счетов с фильтром по датам транзакицй(интервал) ,подразделению ( по номеру/любой), типу, дате открытия
	Кнопка добавить счёт
Переходы: в рез. таблице
	Клиент
	Тип счёта
	Счет
3. Тип счёта
Содержимое:
	Информация о данном типе счёта
Переходы:
	Нет
4. Счет
	Информация о счете
	Последние  N транзакций
	Кнопка редактировать
	Кнопка удалить
Переходы:
	Клиент
	Отделение
	Транзакция
	Тип счета
5. Транзакция
Содержимое: (номер счета известен по странице с которой был совершен переход)
	выбор типа начисление , списание
	сумма
	описание
6.   Отделения
Содержимое:
	Поиск отделения с фильтром(имя, адрес) 
	Кнопка добавить отделение
6.   Отделение
Содержимое:
	Информация об отделении
	Кнопка редактировать
	Кнопка удалить
7. Клиенты
Содержимое:
	Форма для поиска клиентов с фильтром (по  имени, контактной информации, виду счёта, отделению, датам открытия счетов)
	Кнопка добавить клиента
Переходы:
	Клиент
8. Клиент
Содержимое:
	Информация о клиенте
	Список счетов
	Кнопка редактировать
	Кнопка удалить
Переходы:
	Cчет(по номеру из списка)
	Отделение

Схема использования:
1 Получение списка клиентов, в т.ч. по типам, видам счетов в заданном интервале времени и пр.
	Главная страница-> клиенты (вид счета из таблицы счёта, d)
2 Получение списка счетов по их видам, списаниям/начислениям за заданный период
	Главная страница -> счета (вид из таблица счетов, списания/начисления из таблицы транзакций)
3 Получение списка отделений, в. т.ч. по клиентам, счетам и пр.
	Главная страница-> отделения (если ищем по имени адресу)
	Главная страница -> клиенты -> клиент -> отделение
	Главная страница -> счёта -> счёт -> отделение
4. Оформление списания/начисления, включая автоматический учет процентов
	Главная страница -> счета -> счет-> Транзакция
	Главная страница -> клиенты-> клиент -> cчет-> Транзакция
	Для каждого счета уменьшаем счетчик дней до выплаты процентов на 1 , если 0 ставим значение счетчика из типа счета
5. Заведение счета и его закрытие, чтение данных о нем
	Главная страница->Поиск счета (добавить счёт, -> cчет закрыть , читаем из выведенной таблицы)
6. Добавление и удаление данных о клиенте, чтение и редактирование данных о нем
	Главная страница->Поиск клиента-> (добавить, -> клиент Информация / редактировать / удалить)

7.Добавление и удаление отделения, чтение и редактирование данных о нем
Главная страница -> Отделения (добавить, ->отделение чтение / редактировать / удалить )
